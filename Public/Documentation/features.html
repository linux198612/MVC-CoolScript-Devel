Feature List

  • Route-Based System: All routes defined in routes.php
  • Automatic Template Detection: Checks for subdirectory templates first, then global template
  • Bootstrap 5 Integration: Pre-configured for rapid development
  • PSR-4 Autoloading: For App and System namespaces
  • View Rendering: Automatic template/layout wrapping with $viewContent variable
  • 404 Fallback: Automatic fallback to 404.php for missing views
  • Configuration System: Centralized settings in settings.php
  • Database Support: Optional database configuration
  • Clean URLs: Via .htaccess URL rewriting
  • Secure Access: Only public folder is web-accessible

Framework Structure

  • Controllers: Handle business logic, extend System\Core\Render
  • Views: Template files with optional subdirectory-specific layouts
  • Routes: Centralized route definitions
  • Configuration: Separate files for settings, routes, and database

What's New in Coolscript MVC 0.02

  • SQL injection protection: All database queries use prepared statements via Database::query($sql, $params).
  • Session fixation protection: Session ID regeneration in Session.php.
  • Validation: Validator class for basic and extended data validation rules.
  • Flash messages: Flash helper for session-based notifications.
  • View cache: Toggleable view caching for faster rendering.
  • Route cache: Toggleable route caching for faster route loading.
  • Logging: Log levels (error, warning, info, debug) in ErrorLogger.
  • XSS protection: Xss helper for safe HTML escaping.
  • Database connection pooling: Singleton pattern for DB connections.
  • View cache clearing: Render::clearViewCache() method.
  • Request helper: Request class for centralized request data access.
  • Response helper: Response class for simplified JSON responses.
  • Config helper: Config class for centralized configuration management.
  • CSRF token automatic injection: CSRF tokens injected into all <form> elements during view rendering.
  • Changelog tracking: See CHANGELOG.md for all notable changes.

Changed

  • Database layer refactored for security and maintainability.
  • CSRF token automatic injection in forms.
